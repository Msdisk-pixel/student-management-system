# Student Management System

This project is a simple **Student Management System** developed using Python as part of the **SEN 201 assignment**.

## Project Description
The Student Management System is a console-based application that allows users to:
- Add student records
- View stored student records

Each student record contains:
- Student Name
- Matric Number

## Files in This Repository
- `student_management_system.py` – Main Python program
- `SDLC.md` – Software Development Life Cycle (SDLC) documentation

## How to Run the Program
1. Ensure Python is installed.
2. Run the file:
   "student_management_system.py"
4. 3. Follow the on-screen menu instructions.

## Features
- Add new students
- View all students
- Simple and easy-to-use interface
